# 🐾 Pet Care Website Template

## ✨ Features

- Responsive design using Bootstrap 4
- Animated image carousel
- Booking form with date and time picker
- Service descriptions with icons
- Blog and testimonials section
- Google Fonts and Font Awesome integration
- Smooth scroll and scroll-to-top button

## 🛠️ Technologies Used

- HTML5 & CSS3
- JavaScript (Vanilla)
- Bootstrap 4
- PHP
- MongoDB Atlas
- Font Awesome & Flaticon
- OwlCarousel, Tempus Dominus

## 🚀 Setup Instructions

1. **Clone the repository** or download the ZIP file.
2. **Open `index.html`** in any modern web browser to preview the static site.
3. **Customize** the `.html` and `.css` files as needed for your project.

## ⚡ Run Instructions (Dynamic Booking Form)

### 1. Install PHP and Composer

- [Download PHP](https://windows.php.net/download/)
- [Download Composer](https://getcomposer.org/download/)

> Ensure PHP is added to your system PATH so you can run `php` from the terminal.

---

### 2. Install MongoDB PHP Library

Open your terminal and navigate to your project folder:

```bash
cd pet-care-website-template
composer require mongodb/mongodb
```

---

### 3. Start the PHP Development Server

Run the following command inside your project directory:

```bash
php -S localhost:8000
```

Then, open [http://localhost:8000](http://localhost:8000) in your browser.
